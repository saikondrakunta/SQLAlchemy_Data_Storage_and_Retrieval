Analysis of Hawaii Weather Data

Using Python and SQLAlchemy, an analysis of weather data was preformed using data sets collected from Hawaii Weather Stations.

The following links present some of queries

  • Date and Percipitation for Last 1 Year: /api/v1.0/precipitation

  • List of Weather Stations: api/v1.0/stations

  • List of Observed Temperatures (ToBs) for Last 1 Year: /api/v1.0/tobs

  • Minimum, Maximum, and Average Temperatures for a Given Start Date (YYYY-MM-DD): /api/v1.0/

  • Minimum, Maximum, and Average Temperatures for a Given Start Range (YYYY-MM-DD): /api/v1.0//
